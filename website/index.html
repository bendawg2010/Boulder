Before you download

Free + open source · MIT licensed. macOS will say "Boulder can't be opened" the first time — Apple charges $99/year for a Developer ID and Boulder is free. Inspect the source on GitHub.

  1. Drag Boulder.app to Applications
  2. Open it once → macOS pops "can't be opened" → hit Done
  3. System Settings → Privacy & Security → scroll → Open Anyway
  4. Reopen → confirm "Open" → Boulder lives in your menubar 🪨

You only do this once. Future updates open normally.
